从 1.x 换到 2.x¶

SDK 如果要从 1.x 换到 2.x 版本，你需要做的几件事：

1） 安装 SDK 2 ，见 INDEMIND SDK 。

2） 升级固件到 2.x 版本，见 MYNT® EYE 固件 。

3） SDK 1.x 运行后，图像标定参数会保存到 <MYNTEYE_SDK_ROOT>/settings/SN*.conf 。请按照 写入图像标定参数 将 SN*.conf 写入设备。
